# ITR Desk Pro Website Upgrade — Upload Ready

This folder contains the upgraded website. You only need to upload all files and folders to your existing GitHub repository root.

## Upload steps
1. Download and extract this ZIP.
2. Open: https://github.com/siddharthvbhatia-blip/itrdesk
3. Click **Add file → Upload files**.
4. Drag and drop **all files and folders** from the extracted folder:
   - index.html
   - calculator.html
   - checklist.html
   - privacy.html
   - all new SEO pages
   - assets folder
   - sitemap.xml
   - robots.txt
   - 404.html
5. Click **Commit changes**.
6. Wait 1–3 minutes.
7. Open: https://siddharthvbhatia-blip.github.io/itrdesk/
8. Press Ctrl + F5.

## Important
- Do not delete the Google verification HTML file already present in GitHub.
- Extra old files like root script.js or styles.css are harmless.
- After upload, go to Google Search Console and submit sitemap.xml again.
