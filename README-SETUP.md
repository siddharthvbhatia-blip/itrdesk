# ITR Desk Website - Free Static Website

This website is built with plain HTML, CSS and JavaScript. It does not need paid hosting, paid plugins or a backend.

## Files included

- `index.html` - Home page and WhatsApp enquiry form
- `checklist.html` - ITR document checklist page
- `privacy.html` - Privacy policy page
- `assets/styles.css` - Website design
- `assets/script.js` - Mobile menu and WhatsApp enquiry automation
- `robots.txt` and `sitemap.xml` - SEO helper files

## What to edit before publishing

1. In all files, replace `CA Siddharth Bhatia / the applicable CA practice` with your final professional entity name.
2. In `assets/script.js`, replace the WhatsApp number if required:
   `const phone = '917879857126';`
3. In `index.html`, update email/phone if required.
4. In `sitemap.xml`, replace `https://example.com/` with your live website URL.
5. If you later create a Google Form or secure document upload link, add that link in the enquiry section.

## Free hosting option 1: GitHub Pages

1. Create a free GitHub account.
2. Create a new public repository named `itrdesk`.
3. Upload all files from this folder to the repository.
4. Go to Settings > Pages.
5. Under Build and deployment, select `Deploy from a branch`.
6. Select branch `main` and folder `/root`.
7. Save.
8. Your site will usually go live at: `https://yourusername.github.io/itrdesk/`.

## Free hosting option 2: Netlify

1. Create a Netlify account.
2. Go to Add new site > Deploy manually.
3. Drag this complete folder into Netlify.
4. Netlify will provide a free `netlify.app` website URL.

## No-cost enquiry automation

The form does not store data on a server. It creates a ready-made WhatsApp message and opens WhatsApp. This keeps the first version free and simple.

## Compliance notes

- Do not add public fee charts.
- Do not add testimonials/refund screenshots.
- Do not write guaranteed refund, best CA, cheapest filing, or 100% notice-free.
- Do not ask users to share passwords through social media DMs.
- Keep the website factual, educational and professionally dignified.
